## Estrutura do Repositório

Seguindo a organização do template original e as necessidades específicas desta 
pesquisa, os arquivos estão estruturados da seguinte forma:

```text
qualificacao_edvaldo_rezende/
├── apresentacao_seminario/     # Arquivos da apresentação para a disciplina de Seminários de Qualificação
├── dados/                      # Bases de dados
│   ├── brutos                  # Base de dados originais
│   ├── limpos                  # Base de Dados processados
├── produto_bibliografico/      # Arquivos do produto bibliográfico
├── produto_tecnico/            # Arquivos do produto técnico
├── relatorios/                 # Relatórios 
├── scripts/                    # Scripts R 
├── src/                        # Arquivos-fonte do documento Quarto
│   ├── exame_qualificacao.qmd  # Arquivo principal do projeto
│   ├── pre_textuais.tex        # Configurações de capa e folha de rosto
│   ├── referencias.bib         # Base bibliográfica BibTeX
│   └── associacao-brasileira-de-normas-tecnicas-ipea.csl  # Estilo ABNT
├── README.md                   # Documentação e registro de evolução
├── README_Washington.md        # Arquivo original de orientação do Prof. Washington
└── qualificacao_edvaldo_rezende.Rproj  # Arquivo de projeto do RStudio
```

### Detalhamento dos Arquivos e Pastas:
A estrutura do repositório está organizada da seguinte forma:

* **apresentacao_seminario/**: Materiais e slides das apresentações de 
Seminários de Qualificação.
* **dados/**: Bases de dados divididas entre arquivos originais (**brutos/**) e 
tratados (**limpos/**).
* **produto_bibliografico/**: Materiais do artigo científico e registros de 
submissão.
* **produto_tecnico/**: Materiais da solução técnica proposta (protótipos, 
ferramentas ou dashboards).
* **relatorios/**: Relatórios e documentos relevantes do projeto.
* **scripts/**: Códigos em R para automação de coleta, limpeza e análise de 
dados.
* **src/**: Núcleo do projeto em Quarto, contendo o arquivo principal 
(`exame_qualificacao.qmd`), referências bibliográficas (`.bib`), estilo 
ABNT (`.csl`) e configurações de capa (`.tex`).
* **README_Washington.md**: Instruções originais do template fornecido pelo 
Prof. Washington.
* **qualificacao_edvaldo_rezende.Rproj**: Arquivo de configuração do projeto 
no RStudio.
